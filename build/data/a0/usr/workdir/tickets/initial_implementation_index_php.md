# Ticket: Initial Implementation of index.php

## Description
Implement a basic index.php file that serves as a landing page for the application.

## Requirements
- Display a friendly greeting (e.g., 'Hello from Agent Zero!').
- Display a 'System Status' section.
- The system status must include:
  - PHP Version
  - Current Server Time
  - Memory Usage

## Technical Details
- Use standard PHP functions (phpversion(), date(), memory_get_usage()).
- Basic HTML5 boilerplate structure.

## Acceptance Criteria
- Page loads without errors.
- All status metrics are visible and accurate.
